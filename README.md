# AI News Summarizer

AI News Summarizer is a full-stack web application that generates personalized, AI-powered summaries of news articles. The app extracts article content and processes it using Google Gemini.

## 🛠 Tech Stack

**Frontend:** React, Vite, Tailwind CSS  
**Backend:** Java, Spring Boot  
**AI Integration:** Google Gemini API  
**Web Scraping:** Jsoup  
**Build Tool:** Maven  

This project is currently in active development.
